# 镜像人生（MirrorLife）

开放世界式虚拟社会游戏 — 输入现实片段，观察你的分身在虚拟社会中如何反应与成长。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

浏览器会自动打开 `http://localhost:4173/game.html`。

```bash
# 生产构建
npm run build

# 预览构建结果
npm run preview
```

## 技术栈

- 纯前端 Canvas 渲染（HTML5 Canvas 2D）
- 回合制社会模拟引擎
- Vite 构建工具
- MIT 开源协议

## 当前运行模型

这是一个纯前端版本，不依赖服务端，但页面并非静态展示。  
它持续运行一个本地社会状态机：人物有年龄、职业、区域、关系与情绪状态；场景支持自动时间推进（白天/夜间）、阶段循环、环境事件与规则治理，伴随事件流、任务和指标变化。

社会会按回合自动演化；只要“自动演化”开启且社会已初始化，世界会持续产生日志与可视化变化。

## 已实现模块

1. 镜像舱：输入现实困惑，选择镜子、旁观、陪伴三种分身回应模式。
2. 人生剧本：录入身份、关系、反复模式和分身禁区，数据保存在本地浏览器。
3. 交换人生：体验匿名重构的人生片段，并把回声写入档案。
4. 精神漂流瓶：投放未说出口的话，等待非强制、偶发式回声。
5. 回家模式：模拟现实情感陪伴机器人对虚拟体验的承接。
6. 回声档案：保存最近的镜像对话、交换人生和漂流回声。
7. 虚拟社会：输入一句现实场景后，分身自动运转并产出事件流、任务看板与社会准则透明指标。
8. 安全治理：敏感内容触发本地高风险提示与分流路径；本地按钮可清空全部数据。

新增动态化增强（v0.4）：

1. 世界阶段循环：开放晨会 / 协作高峰 / 张力窗口 / 修复夜按回合切换。
2. 环境事件系统：每 3 回合触发公开发言、修复、摩擦等事件，影响情绪、信任和张力。
3. 指标趋势：自由、平等、开放指标支持趋势显示（每回合记录历史）。
4. 社会总览：新增“当前阶段”与“自动演化”状态，便于判断世界是否持续运行。

本地“用户旅程完整性检查”建议：

1. 打开页面后先切到“人生剧本”，填写身份/关系/模式，点击“保存剧本”。
2. 回到“镜像舱”，输入现实片段并点击“交给分身”，确认镜像回声和社会事件流变化。
3. 切到“交换人生”，点击至少 2 张卡片，确认回声档案条目增长。
4. 切到“虚拟社会”，点击“重置并生成社会”，确认“自动演化”开启后每数回合都有新事件；必要时可手动暂停/恢复。
5. 在“虚拟社会”点击一次玩家动作（提案/协作/安抚/倾听），确认事件流有新增记录。
6. 切到“漂流回声”，投放文本后点击“等待回声”，观察回声和社会回写。
7. 切到“回家模式”，切换三种陪伴模式并确认文案变化。
8. 点击“清空本地数据”后刷新页面，确认回声档案、场景与脚本输入可重置。
